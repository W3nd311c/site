Aula 5: Semáforo com Arduino

Funcionamento do Semáforo

Na aula 5, vamos aprender como construir um semáforo funcional com Arduino. Este semáforo terá duas vias, cada uma com suas próprias luzes de sinalização (vermelha, amarela e verde).

Detalhamento do Circuito

Explicação do Programa

Aqui está o código utilizado para controlar o semáforo:


        
        
            // C++ code
//
int i = 0;

void setup()
{
  pinMode(2, OUTPUT);
  pinMode(3, OUTPUT);
  pinMode(4, OUTPUT);
  pinMode(8, OUTPUT);
  pinMode(9, OUTPUT);
  pinMode(10, OUTPUT);
}

void loop()
{
  digitalWrite(2, HIGH);
  digitalWrite(3, LOW);
  digitalWrite(4, LOW);
  digitalWrite(8, HIGH);
  digitalWrite(9, LOW);
  digitalWrite(10, LOW);
  delay(5000); // Wait for 5000 millisecond(s)
  digitalWrite(9, HIGH);
  digitalWrite(3, LOW);
  digitalWrite(8, LOW);
  delay(2000); // Wait for 2000 millisecond(s)
  digitalWrite(2, LOW);
  digitalWrite(3, LOW);
  digitalWrite(4, HIGH);
  digitalWrite(8, LOW);
  digitalWrite(9, LOW);
  digitalWrite(10, HIGH);
  delay(5000); // Wait for 5000 millisecond(s)
  digitalWrite(9, LOW);
  digitalWrite(3, HIGH);
  delay(2000); // Wait for 2000 millisecond(s)
}

Demonstração Prática

Atividade Prática

Para praticar, tente modificar o tempo de cada fase do semáforo ou adicionar um botão de pedestres para interromper o tráfego.
